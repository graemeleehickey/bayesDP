# bdpbinomial Object Print

`print` method for class `bdpbinomial`.

## Usage

``` r
# S4 method for class 'bdpbinomial'
print(x)
```

## Arguments

- x:

  object of class `bdpbinomial`. The result of a call to the
  [`bdpbinomial`](https://graemeleehickey.github.io/bayesDP/reference/bdpbinomial.md)
  function.

## Details

Returns same output as a call to
[`summary`](https://graemeleehickey.github.io/bayesDP/reference/summary-bdpbinomial-method.md).
